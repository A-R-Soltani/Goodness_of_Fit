# Goodness_of_Fit
Goodness of fit test using the standard normal critical values, appeared in the article: A pivot function and its limiting distribution: applications in goodness of fit and testing hypothesis;  https://doi.org/10.1080/02331888.2019.1667360


The file "Soltani-Goodness-of-Fit.R" is the R program to apply the Soltani Goodness of Fit Test.
The file "Soltani-GoF_test.R" Simulates values for the test statistics and shows its asymptotic standard normal distribution.
